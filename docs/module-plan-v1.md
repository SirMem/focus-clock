# 专注时钟 — 模块规划 v1

> 归档日期：2026-06-19
> 状态：第一版规划，待评审

---

## 一、产品定位

一款**微信原生 + AI 驱动的个人效率教练**，基于番茄工作法，围绕
**「计划 → 执行 → 反思 → 优化」**闭环，帮助用户在微信生态内实现专注管理、
任务规划和自我成长。

## 三、模块详情

### 模块 1：番茄时钟（升级已有）⏱️

当前状态：已有基础计时、白噪音。

| 升级项 | 说明 | 优先级 |
|--------|------|--------|
| 专注/休息计时核心 | 保持现有功能 | P0 |
| 自定义任务名 + 绑定待办 | 开始专注时从待办列表选一个任务关联 | P0 |
| 背景白噪音 | 保持现有，可扩充音源 | P1 |
| 干扰记录 | 专注中断时记录原因（消息/电话/走神等） | P1 |
| AI 智能时长建议 | 根据历史数据推荐最优专注/休息时长 | P2 |

---

### 模块 2：待办清单（升级已有）📋

当前状态：已有基础增删改、标记完成。

| 升级项 | 说明 | 优先级 |
|--------|------|--------|
| 基础列表增删改勾 | 保持现有 | P0 |
| 四象限优先级矩阵 | 重要/紧急、重要不紧急、紧急不重要、不重要不紧急 | P1 |
| AI 智能拆解 | "写论文" → AI 拆成 5 个番茄钟可完成的小步 | P1 |
| 专注关联 | 待办与番茄钟绑定，统计完成耗时 | P0 |

---

### 模块 3：统计看板（升级已有）📊

当前状态：已有环形图统计专注/休息次数。

| 升级项 | 说明 | 优先级 |
|--------|------|--------|
| 日/周/月趋势图 | 专注时长趋势、完成率变化折线图 | P0 |
| 效率高峰分析 | "你周三上午效率最高" | P1 |
| 情绪-效率关联图 | 结合日记情绪标签，分析情绪对效率影响 | P1 |
| 连续打卡日历 | GitHub 风格绿点图 | P0 |

---

### 模块 4：日记反思（新增）📝

AI 反馈的数据基础，也是用户"复盘"的入口。

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 每日一记 | 自由文本记录今日感受 | P0 |
| 情绪标签 | 😊😐😢 快速标记心情 | P0 |
| 番茄自动关联 | 自动关联当天完成的专注任务 | P0 |
| AI 引导提问 | "今天哪个番茄最难坚持？为什么？" | P1 |
| AI 周报 | 自动生成每周效率总结 + 情绪趋势 | P1 |

---

### 模块 5：AI 教练（新增）🤖

核心差异化模块，整个应用区别于其他番茄钟的关键。

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 每日效率评分 | 基于专注完成率 + 待办完成率 + 情绪的综合评分 | P0 |
| 智能趋势洞察 | "你这周比上周专注时长提升 15%" | P0 |
| 个性化建议 | "建议明天把最难的任务放在 9-11 点" | P1 |
| 习惯分析 | "已连续专注 7 天，但休息严重不足" | P1 |
| AI 对话入口 | 用户直接问"我最近效率怎么样？" | P2 |

---

### 模块 6：目标 & 排行榜（新增）🎯

增强长期粘性和社交激励。

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 个人目标设定 | 本月专注 50 小时、完成 30 个待办 | P1 |
| 连续打卡记录 | 最长连续专注天数 | P1 |
| 好友排行榜 | 基于专注时长的好友排名 | P2 |
| 分享 AI 周报 | 生成精美卡片分享到朋友圈 | P1 |

## 四、用户界面结构（Tab Bar 规划）

```
底部 Tab（4 个）
┌──────────┬──────────┬──────────┬──────────┐
│  专注     │  待办     │  统计     │  我的     │
│  (番茄钟)  │  (清单)   │  (看板)   │  (设置+)  │
├──────────┼──────────┼──────────┼──────────┤
│          │          │ 日记入口  │ AI 教练   │
│          │          │          │ 目标      │
│          │          │          │ 排行榜    │
│          │          │          │ 设置      │
└──────────┴──────────┴──────────┴──────────┘
```

- **专注**：番茄钟计时为主页
- **待办**：待办列表 + 四象限
- **统计**：统计看板 + 日记入口
- **我的**：AI 教练 / 目标 / 排行榜 / 设置 / 关于

## 六、技术方向

| 维度 | 方案 |
|------|------|
| 前端框架 | 微信小程序原生 |
| AI 能力 | Grok API（通过已有服务器代理） |
| 数据存储 | 微信本地存储 + 后续可扩展云开发 |
| 图表 | wxcharts（已有）/ 可按需升级 |
| 日记 | 本地存储 Markdown 格式 |
