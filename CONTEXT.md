# 专注时钟 Context

本上下文描述“专注时钟”产品中的核心领域语言，帮助区分身份、资料与专注行为等概念。

## Language

**静默身份登录**:
用户点击登录后，系统只确认当前微信用户身份并进入应用，不在登录过程中采集头像或昵称。
_Avoid_: 一键授权资料登录、弹窗填写登录信息

**用户资料完善**:
用户在已登录状态下主动补充或更新头像、昵称等展示资料。
_Avoid_: 登录资料授权、登录信息填写

**微信用户**:
未完善资料时展示的默认用户昵称，表示系统已识别身份但尚未获得用户自定义展示资料。
_Avoid_: 匿名用户、游客

## Example dialogue

开发者：点击登录后要不要弹窗让用户填头像昵称？
领域专家：不要。登录是静默身份登录，只负责进入应用。
开发者：那昵称默认显示什么？
领域专家：默认显示“微信用户”。用户需要个性化资料时，再走用户资料完善流程。
开发者：头像昵称在哪里更新？
领域专家：在已登录后的资料页或资料完善入口中更新，不阻塞首次登录。
